A desktop reference for

Advanced Dungeons & Dragons
2nd Edition

Rules Reference  &  Campaign Tools

The full 2e rulebooks, quick-reference DM & actions screens, an interactive character builder, the complete spell compendium, and a local-AI rules assistant — searchable, offline, on your desktop.

⬇  Download for Windows
Windows 10/11 · 64-bit · ~130 MB zip · no install needed  ·  Source on GitHub

What's inside

📚 Full rulebooks

PHB, DMG, Monstrous Manual and more — browsable, with fast full-text search.

⚔️ DM & Actions screens

50+ quick-reference tables and every combat action, grouped and filterable.

🧙 Character Builder

Build a character step by step — abilities, race, class, proficiencies, equipment, and spells — with your house rules applied.

📖 Spell Compendium

Every wizard & priest spell, searchable and filterable by class, level, and school.

🎲 Roll20 export

Send a finished character to a house-ruled Roll20 sheet — stats, saves, gear, and spells in one paste.

✧ Jarvis (local AI)

Ask rules questions in plain English — answered from the books, citing the pages.

🧮 House-rule tools

Your house rules baked in, plus a floating THAC0 / AC converter.

🗂️ Browser-style

Tabs, bookmarks, back/forward, zoom, and find-on-page — it reads like a browser.

Getting started

  1. Download the zip and extract it anywhere (Desktop, Documents…).
  2. Open the folder and run DnD2eRules.exe. No installer, nothing to set up.
  3. Browse, search, and use the screens right away — everything works offline.
Optional — for the “Jarvis” AI assistant: install Ollama (free) and run ollama pull llama3.1. Jarvis runs the model locally on your machine — no account, no API key, no cost.
